# vue2-checking

## Local environment
```
$ git clone https://github.com/kdevsoftware/vue2-checking.git
$ cd vue2-checking
$ npm install
$ npm run serve
```
Navigate to `http://localhost:8080/`.

## Deploying to github page
```
$ npm deploy
```
Navigate to `https://kdevsoftware.github.io/vue2-checking/`.
